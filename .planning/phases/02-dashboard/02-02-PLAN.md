SECOND useEffect (add after the existing profile fetch useEffect):
```javascript
useEffect(() => {
  setLoading(true);
  fetch(`/api/profiles/${profileId}/telemetry`)
    .then(res => {
      if (!res.ok) throw new Error('Failed to load telemetry data');
      return res.json();
    })
    .then(data => { setTelemetry(data); setLoading(false); })
    .catch(err => { setError(err.message); setLoading(false); });
}, [profileId]);
```

CHART CARD JSX (add after the existing h1 tag):
```jsx
<div style={CARD_STYLE}>
  <p style={{
    fontSize: '0.8rem',
    fontWeight: 600,
    color: '#666',
    marginBottom: '16px',
    letterSpacing: '0.06em',
    textTransform: 'uppercase',
    margin: '0 0 16px 0',
  }}>
    Correction Angle — Past 7 Days
  </p>
  {loading && (
    <p style={{ color: '#666', fontSize: '0.9rem' }}>Loading data…</p>
  )}
  {error && (
    <p style={{ color: '#c62828', fontSize: '0.9rem' }}>{error}</p>
  )}
  {!loading && !error && (
    <div style={{ width: '100%', height: 300 }}>
      <ResponsiveContainer width="100%" height="100%">
        <LineChart data={telemetry} margin={{ top: 8, right: 24, bottom: 8, left: 8 }}>
          <CartesianGrid vertical={false} stroke="#f0f0f0" />
          <XAxis
            dataKey="recorded_at"
            tickFormatter={(v) => new Date(v).toLocaleDateString('en-US', { weekday: 'short' })}
            tick={{ fontSize: 11, fill: '#666' }}
            interval="preserveStartEnd"
          />
          <YAxis
            domain={[5, 130]}
            allowDataOverflow
            tickFormatter={(v) => `${v}°`}
            tick={{ fontSize: 11, fill: '#666' }}
            width={44}
          />
          <Tooltip
            formatter={(v) => [`${Number(v).toFixed(1)}°`, 'Correction Angle']}
            labelFormatter={(v) => new Date(v).toLocaleString()}
            contentStyle={{ fontSize: '0.8rem', border: '1px solid #e0e0e0' }}
          />
          <Line
            type="monotone"
            dataKey="correction_angle"
            stroke="#0D8ABC"
            dot={false}
            strokeWidth={1}
            isAnimationActive={false}
          />
        </LineChart>
      </ResponsiveContainer>
    </div>
  )}
</div>
```

IMPORTANT rendering flags on Line:
- `dot={false}` — prevents ~20k SVG circle elements from being created
- `isAnimationActive={false}` — prevents multi-second animation with large datasets
- `strokeWidth={1}` — keeps the path thin and readable at high point density

The parent div MUST have explicit pixel height (300) — ResponsiveContainer collapses to 0px if its parent has no explicit height.

Remove the existing placeholder paragraph: `<p style={{ color: '#666', marginTop: '8px' }}>Dashboard coming in Phase 2</p>`

Also update the h1 to have `marginBottom: '32px'` for visual separation before the chart card.
